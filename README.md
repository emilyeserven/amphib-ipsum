# amphib-ipsum

Lorem ipsum but make it froggy — a lorem ipsum generator using words associated with frogs and toads.

## About

Amphib Ipsum generates placeholder text from a vocabulary of ~250 frog and toad related words, including:

- **Scientific names** — *Dendrobates*, *Agalychnis*, *Rhinella*, *Xenopus*, and more
- **Common names** — bullfrog, spring peeper, poison dart frog, coqui, glass frog
- **Anatomy terms** — tympanum, parotoid, vocal sac, toe pad, dorsal
- **Habitat words** — riparian, vernal pool, cloud forest, bromeliad, arboreal
- **Behavior terms** — amplexus, metamorphosis, chorusing, aposematic, brumation
- **Connectors** — verbs and prepositions for natural sentence flow

Generate output by paragraphs, sentences, or word count.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
